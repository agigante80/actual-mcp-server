# Scratch Tests

This directory contains exploratory and one-off test scripts that are not part of the regular test suite.

**Purpose**: Quick experiments, debugging tests, and temporary validation scripts.

**Note**: These tests may be out of date and are not maintained. Use for reference only.

## Files

- `test-comprehensive.js` - Comprehensive MCP tool testing
- `test-transaction-*.js` - Various transaction API test approaches
- `test-verify-fields.js` - Field validation tests
- `test-update-tool.mjs` - Tool update testing

These scripts were moved from the root directory during project reorganization on December 10, 2025.
